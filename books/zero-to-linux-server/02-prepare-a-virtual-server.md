## 【メモ】第2章 仮想サーバーを用意しよう
- VPS（Virtual Private Server）
  専用サーバーと仮想サーバーのいいとこどりをしたもの。
  仮想化ソフトウェアを使って仮想的なサーバーを用意し、それを借りる方法。
  利用者にとっては、専用サーバーと同様の自由度がありつつ、価格は専用サーバーを借りるよりも安くなる。（物理的なサーバーを占有するわけではないため）
- VNC（Virtual Network Computing）
  コンピューターの画面を仮想的に別のコンピューターに転送し、操作できるようにする仕組み。
- VNCコンソール
  コンソール（Linuxマシンに直接接続された入出力装置）をWebブラウザ上に再現したもの。
- SSH（[Secure Shell](https://wa3.i-3-i.info/word11722.html)）
  クライアントPCと サーバーの間で安全な通信を行うための仕組み。
  通信する内容を暗号化して送る。
  SSHクライアントで接続すると、ネットワーク経由でLinuxサーバーにログインし、コマンドによる作業を実施できる。
